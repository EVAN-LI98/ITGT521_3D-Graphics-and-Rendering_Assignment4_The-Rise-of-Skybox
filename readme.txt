Keyboard inputs for plane and propeller (subpart):

 ------------------------------------------------------> Plane 1 
     i   = moves the plane forward   
     k   = moves the plane backward   
     q,e = rolls the plane    
     a,d = yaws the plane     
     x,s = pitches the plane     
 ------------------------------------------------------> Plane 2
     I   = moves the plane forward    
     K   = moves the plane backward    
     Q,E = rolls the plane     
     A,D = yaws the plane     
     X,S = pitches the plane    
 ------------------------------------------------------> Plane Propeller 
     r   = Rotates right propeller    
     y   = Rotates left propeller    
     t   = Rotates Front propeller    
     g   = Rotates Subsubpropeller1    
     h   = Rotates Subsubpropeller2    
     H   = Rotates Subsubpropeller3   
 ------------------------------------------------------> Select Camera
     v   = Select camera to view     
     b   = Select camera to control    
 ------------------------------------------------------> Light Control 
     1,3 = translates light up/down (+y/-y)    
     2,4 = translates light (+z/-z)    
     5,6 = translates light (+x/-x)    
     9   = toggles diffuse light on/off      
     8   = toggles specular light on/off     
     7   = toggles ambient light on/off      
 ------------------------------------------------------> Mouse Control 
     Mouse inputs for world-relative camera:
     Hold left button and drag  = controls azimuth and elevation   
     
     (Press CTRL (and hold) before left button to restrict to azimuth control only,
      Press SHIFT (and hold) before left button to restrict to elevation control only)  
 
      Hold right button and drag = controls distance
